# AwesomeChocolateSalesAnalaysis
####This report analyzes the sales performance of Awesome Chocolates, focusing on key metrics such as total sales, shipment count, box volume, and performance by salesperson, product, and region. The dashboard provides insights into trends, best-performing products, and shipment behavior across different time periods.
###Data Sources
####Geo---GeoID,Geo,Region
####People---Salesperson,SPID,Team,Location
####Products---PID,Product, Category, Size,Cost_per_box
####Sales--SPID, GeoID,PID ,SaleDate,Amount ,Customers,Boxes
